# Agentic AI Banking Email Processing System

An end-to-end Agentic AI system for intelligent banking complaint processing.

## Features

- ML-based Email Classification
- Confidence Agent
- Novelty Agent
- Customer Context Retrieval
- Policy Evaluation
- Decision Agent
- Action Agent
- FastAPI Backend
- Streamlit Frontend

## Tech Stack

- Python
- FastAPI
- Streamlit
- Scikit-learn
- Sentence Transformers
- SQLite
- SQLAlchemy

## Architecture

Email
↓
Confidence Agent
↓
Novelty Agent
↓
Context Agent
↓
Policy Agent
↓
Decision Agent
↓
Action Agent
↓
Ticket Generation
